# Code of Conduct

By participating in this project, you agree to uphold a respectful, inclusive,
and harassment-free environment for everyone.

## Basic Guidelines
- Be respectful and constructive.
- Do not harass, insult, or discriminate against others.
- Welcome beginners and encourage learning.
- Focus discussions on the project and technical topics.

## Reporting Issues
If you see unacceptable behavior, open an issue in the repository or
contact the maintainers privately.

This project follows the Contributor Covenant principles.
